# ğŸ“– Bookstore <a name="about-project"></a>

**Bookstore** The Bookstore is a web application that allows users to display a list of books, add a book, and remove a selected book. Users can view a list of books on the homepage of the application. Each book item displays information such as the book title, author, and category. Users can click on a "Add Book" button to add a new book to the list, and they can also click on a "Remove Book" button to remove a selected book from the list.
The Bookstore application uses React for building the user interface, and Redux for managing the application state.
